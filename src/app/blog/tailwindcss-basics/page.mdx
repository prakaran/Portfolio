# Tailwind CSS Basics

Welcome to the world of Tailwind CSS! This guide will walk you through the fundamentals of Tailwind CSS and how to use it in your Next.js application.

## What is Tailwind CSS?

Tailwind CSS is a utility-first CSS framework for rapidly building custom user interfaces. It provides low-level utility classes that can be composed to create complex designs without writing custom CSS.

## Installation

To install Tailwind CSS in your Next.js application, follow these steps:

1. Install Tailwind CSS and its peer dependencies:

```bash
npm install -D tailwindcss postcss autoprefixer
```

2. Generate the Tailwind CSS configuration files:

```bash
npx tailwindcss init -p
```

This will create a `tailwind.config.ts` and `postcss.config.ts` file in your project root.

3. Configure the `content` option in `tailwind.config.ts` to include all files that contain Tailwind classes:

```typescript
import type { Config } from "tailwindcss";

const config: Config = {
  content: ["./src/**/*.{js,ts,jsx,tsx,mdx}"],
  theme: {
    extend: {},
  },
  plugins: [],
};

export default config;
```

4. Add the Tailwind directives to your global CSS file (e.g., `src/app/globals.css`):

```css
@tailwind base;
@tailwind components;
@tailwind utilities;
```

## Basic Usage

Tailwind CSS provides a comprehensive set of utility classes for styling your application. Here are some common examples:

### Text Styling

```html
<h1 className="text-3xl font-bold underline">Hello, Tailwind!</h1>
```

### Background Colors

```html
<div className="bg-blue-500 text-white p-4 rounded-lg">This is a blue box.</div>
```

### Flexbox and Grid

```html
<div className="flex justify-between items-center">
  <div>Left</div>
  <div>Right</div>
</div>
```

### Spacing

```html
<div className="m-4 p-4 border border-gray-300">
  This box has margin and padding.
</div>
```

## Responsive Design

Tailwind CSS provides responsive design utilities that allow you to create responsive layouts easily. You can use breakpoints like `sm`, `md`, `lg`, and `xl` to apply styles at different screen sizes.

```html
<div className="text-base md:text-lg lg:text-xl">
  This text changes size based on screen width.
</div>
```

## Customization

You can customize Tailwind CSS by modifying the `tailwind.config.ts` file. For example, you can add custom colors, fonts, or spacing:

```typescript
import type { Config } from "tailwindcss";

const config: Config = {
  theme: {
    extend: {
      colors: {
        custom: {
          DEFAULT: "#123456",
          light: "#654321",
        },
      },
    },
  },
};

export default config;
```

## Conclusion

Tailwind CSS provides a powerful and flexible way to style your Next.js application. By using its utility-first approach, you can build custom designs rapidly and maintain a consistent codebase.

For more information, check out the [official Tailwind CSS documentation](https://tailwindcss.com/docs).
